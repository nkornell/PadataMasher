Padata Masher

Welcome to Padata Masher! We offer free tools to help you clean and organize your data. We never upload, store, or modify your files.

Combine & Clean. Have a batch of data files to analyze? Run this program. It will give you a report about each file and let you decide which ones to include in your analysis. When you press Export Data it will combine them into a new file with an added column that flags wonky rows for your inspection.

Pivot Table Plus. Want to compute a mean for each of your participants? You could use a pivot table, but this program does more: Compute each participant's median, or standard deviation, Pearson or a gamma correlation, or other stuff. Give it a data file, identify the variables and analysis you want to run, and press Export Results, it's that easy!
